# Laser-Defender
2D Space shooter

### Player experience
    - Frantic
### Core Mechanic
    - Shoot enemies, dodge bullets
### Theme
    - Space wars
### Core game loop
    - Single level, endless play where player shoots
      enemies for points until health reaches zero and
      game over
